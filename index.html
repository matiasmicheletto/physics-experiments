Spring-mass system

A simple spring-mass system on a frictional surface.

Ball on table

A ball on a table attached to a box through a hole in the middle of the table.

Gravitation

A simulation of the movement of the moon around the earth.

Inclined plane

A sliding box over an inclined frictional surface.

Double inclined plane

A pair of boxes sliding on different inclined planes and attached between each other with a string.

Ramp

A box sliding in a frictionless skate ramp with a short rough surface and colliding with a spring.

Pendulum waves

A set of pendulums of different lengths that create a wave effect.

About this software

This is a small compendium of javascript 3D animations that allows the user to perform mechanics experiments with bodies and systems.
